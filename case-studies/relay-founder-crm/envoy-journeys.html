Trigger · Founder
Decision to Hire — Handoff Mode Activated
Founder action

Founder opens the Envoy app and activates Handoff Mode. Envoy begins generating the Pipeline Handoff Report automatically from all stored conversation history.

↓
📊
Automated · Envoy
Pipeline Handoff Report — Auto-Generated
Fully automated

Envoy assembles everything it has captured across all channels — email, Telegram, WhatsApp, voice notes, call transcripts — into a structured, deal-by-deal briefing.

All active deals + full cross-channel history
Open commitments per deal (both sides)
Warmth score + last contact date per deal
Voice memo transcripts + quick notes
AI-generated relationship summary per deal
Deal velocity signals (hot / cooling / cold)
All channels, one report: The report consolidates history across every channel Envoy has monitored — Gmail, Telegram, WhatsApp, Zoom transcripts, voice notes. The hire doesn't need to know which channel each conversation happened on.
↓
✍️
Founder Review
Annotate and Assign Deals
~30 min review

Founder reviews the report, adds human context where needed, and makes assignment decisions. Envoy has the data — the founder adds the judgment.

Mark deals: High / Medium / Low priority
Add context notes on key deals
Flag deals to keep personally (sensitive / strategic)
Assign remainder to new hire
↓
🔑
Handoff Moment
Share Workspace Access
Single action

Founder shares the Envoy workspace with the new hire's email. New hire receives an invite and joins the workspace with scoped access to their assigned deals.

Ongoing · New Hire
On-Demand Context via Envoy Agent
Natural language · Any time

"Brief me on TechCorp"
"What did we last discuss with Priya?"
"What commitments are open on this deal?"
"Which deals should I prioritise this week?"
Response time: Agent returns full relationship summary in < 3 seconds. No searching across apps. No asking the founder. The context travels with the deal.
